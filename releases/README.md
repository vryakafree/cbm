### 🚀 Installation:
1. **Download**: `customcobblemonmusicmod-1.0.0.jar`
2. **Requirements**: Minecraft 1.21.1 + Cobblemon 1.6.1+ + Fabric
3. **Install**: Place in `mods` folder
4. **Test**: `/cobblemusic status` in-game
